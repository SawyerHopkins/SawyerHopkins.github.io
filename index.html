Projects

SciLib

SciLib is a numerical analysis and computation engine created using C#, F#, and XNA. This project started as an experiment to explore the logic needed to perform generic N-Space calculations while also learning the oddities of Microsoft's new scientific programming language F#. Later I found that having this N-Space computational engine could be very useful for various elements of my research and course work. Today SciLib is a compilation of various scientific and mathematical tools I have built using this computational engine and includes basic mathematics, integration, RK4 analysis, and 3D plotting amongst various other tools. The source code is currently available on my GitHub which is accessible via the social links on the left.
